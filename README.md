# Pokémon Swipe

Swipe through Pokémon and track your favorites. Each Pokémon includes a Chuck Norris joke.

## Quick Start

**Windows:**
```bash
start.bat
```

**macOS/Linux:**
```bash
chmod +x start.sh
./start.sh
```

## Requirements

- Node.js 18+
- PostgreSQL 14+

## Manual Setup

1. Install dependencies:
```bash
npm install 
```

2. Start backend:
```bash
cd apps/api
npm run start:dev
```

3. Start frontend (new terminal):
```bash
cd apps/web
npm run dev
```

4. Open http://localhost:3000
